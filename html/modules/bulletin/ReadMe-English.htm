About This Module

This module is a kind of news modules. I started to develop this module purpose to renew the old source code. Ultimately, I took in some usefull hacks and reallized duplication by copying and renaming directory.

Features

  • To post, edit and delete articles
  • Archive
  • RSS
  • Duplication of module
  • To import stories from "news"
