A propos de ce Module

Ce module est une sorte de module news. J'ai commencé par développer ce module pour renouveller l'ancient code source. Finalement, j'ai pris quelques hacks utiles et réalisé la possibilité de clonage/duplication simplement en copiant et renommant le répertoire.

Fonctions

  • Publier, editer et effacer articles
  • Archive
  • RSS
  • Clonage/Duplication du module
  • Importation de Sujets et Articles depuis "news"
